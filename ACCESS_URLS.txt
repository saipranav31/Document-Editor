🔧 TROUBLESHOOTING:

Problem: "This site can't be reached"
Solution: Run the firewall setup script (see step 2 above)

Problem: Different WiFi network
Solution: Connect phone to same WiFi as computer

Problem: IP address changed
Solution: Run "ipconfig" in terminal, find new IPv4 Address
         Update the phone URL with new IP
